unevent

Something about

Attach an event handler function for one or more events to the selected elements if the event was not triggered for a given interval.

This is useful if you want to fire a callback only after a delay, like the resize event, or else.

How to use

It is quite simple!

1st Include jQuery and the plugin:

<script src='jquery.js'></script>
<script src='jquery.unevent.js'></script>

2nd Initialize unevent like any other on-event handler, except that you can pass an extra parameter as a last:

$(window).on('resize', function(e) {
    // console.log(e.type + ' was 250ms not triggered');
}, 250);

3rd Trigger your event!

Example

Move your mouse and wait 250ms...

$('#example').on('mousemove', function(e) {
    $('#box').html(e.type + ' was 250ms not triggered');
}, 250);
